# HorizonOS

**Author:** Riyan

## Vision
HorizonOS is a hobby operating system built from scratch to learn operating system development and create innovative system software.

## Long-term Goals
- Build a custom kernel
- Create a fast and efficient scheduler
- Design a modern file system
- Develop a beautiful desktop environment
- Explore AI-powered operating system features
- Learn every major component of an operating system
